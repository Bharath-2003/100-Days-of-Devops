# 100 Days of DevOps - KodeKloud

This repository contains my learning journey through the 100 Days of DevOps course from KodeKloud.

## 📚 Structure

```

├── README.md
├── days/
│   ├── day-001/
│   │   ├── notes.md
│   │   ├── answers.md
│   │   └── resources.md
│   ├── day-002/
│   └── ...
└── templates/
    ├── notes-template.md
    ├── answers-template.md
    └── resources-template.md
```

## 📖 How to Use

Each day's content is organized in its own directory under `days/`:
- **notes.md**: Key concepts and learnings from the day
- **answers.md**: Solutions to exercises and challenges
- **resources.md**: Additional resources, links, and references

## 🚀 Quick Start

To add a new day's content:
1. Create a new directory: `mkdir -p days/day-XXX`
2. Copy templates: `cp templates/*.md days/day-XXX/`
3. Fill in your notes, answers, and resources
4. Commit your changes: `git add . && git commit -m "Add Day XXX content"`
